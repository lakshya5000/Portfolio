Network Traffic Visualization And Analysis Tool

A tool designed to capture, visualize, and analyze network traffic to better understand network behavior and performance. It provides insights into traffic patterns, potential bottlenecks, and anomalies, demonstrating practical application of computer networking concepts.

A cloud-deployed intrusion detection system designed to monitor and secure cloud environments. It analyzes network and system activity to identify potential threats, showcasing practical implementation of cloud architecture, scalable design, and cybersecurity principles.

SOS Alert System

A gesture-based SOS Emergency Alert System that detects a predefined hand signal using a webcam. On detection, it automatically records audio and video evidence, retrieves the user’s approximate real-time location, and sends an alert email to emergency contacts with all details. The system enhances personal safety by enabling quick, touch-free emergency communication.
